#  UberX × SpaceX

## Descripción

UberX × SpaceX es una aplicación web desarrollada con **HTML5** y **CSS3** que simula un servicio de turismo espacial. El usuario puede explorar destinos, seleccionar un viaje, elegir un método de pago y visualizar el recibo de la reserva.

Este proyecto fue desarrollado como parte de un reto de maquetación, utilizando únicamente tecnologías nativas, sin JavaScript ni frameworks.

---

##  Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Variables CSS
- Animaciones CSS

##  Características

- Diseño responsive para dispositivos móviles y escritorio.
- Navegación entre pantallas mediante enlaces HTML.
- Interfaz moderna inspirada en UberX y SpaceX.
- Uso de Flexbox y Grid para la distribución de elementos.
- Animaciones y transiciones realizadas únicamente con CSS.

---

## ▶Cómo ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Ejecutar el proyecto con la extensión **Live Server**.
4. Navegar por las diferentes pantallas.
